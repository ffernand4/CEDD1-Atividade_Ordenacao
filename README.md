# CEDD1-Atividade_Ordenacao

## Método escolhido: Insertion Sort

O Insertion Sort percorre o vetor da esquerda para a direita, inserindo cada elemento na posição correta entre os elementos já ordenados. O algoritmo compara o elemento atual com os anteriores e desloca os maiores para a direita até encontrar a posição adequada. Após a execução, o vetor fica ordenado em ordem crescente.

### Vetor original

49 38 58 87 34 93 26 13

### Vetor ordenado

13 26 34 38 49 58 87 93
